# fenxiang-money-skill

> 发一个商品链接，帮你查优惠券、全网比价、看历史价格走势，告诉你现在该不该买。

## 核心能力

- **商品转链** — 自动查找隐藏优惠券，生成带券推广链接，显示到手价和佣金
- **跨平台比价** — 同一商品在多个平台的价格对比，找出全网最低价
- **历史价格分析** — 价格走势 + 智能购买建议（破历史低 / 价格适中 / 建议等降价）

## 支持平台

淘宝 · 天猫 · 京东 · 拼多多 · 抖音 · 唯品会 · 美团

## 使用方式

直接发商品链接或淘口令即可，无需任何指令：

```
https://u.jd.com/xxx
```

也可以明确意图：

```
这个链接帮我比价
这个商品值不值得买
帮我看看历史价格
```
